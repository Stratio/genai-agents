# Stratio Data — Data MCPs Reference

Load this skill when you are going to work with Stratio governed data: query domains, explore tables, obtain data via MCP, or propose business knowledge. Contains the mandatory rules, usage patterns and best practices for all data MCP tools (query_data, list_domains, search_domains, generate_sql, etc.).

**When to invoke**: Before the first interaction with any data MCP tool in a conversation, or when you need to refresh usage rules (immutable domain_name, MCP-first, post-query validation, clarification cascade).

Read `skills-guides/stratio-data-tools.md` for the complete reference.
